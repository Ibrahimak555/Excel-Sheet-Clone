This is Excel/Sheets Clone with basic text editing features.

1) The Saving and uploading format is done in 'JSON'.

2) The Grid consists of 100 rows and 26 columns (A-Z), and each cell having its own name and property. Eg: D6.

3) Multiple cells can be selected using 'ctrl'. Eg: Control select cells from E3 to H8 (Represented as E3:H8).

4) Copy, Paste and Cut features are applicable for only "Selected cells" (Dotted cells).

5) Create new sheets using 'Add sheets icon' and right-click to delete the sheets.

6) Use formula bar to access basic arithmetic functions (SUM, SUB, MUL). To get the sum of A1 and A2 in the cell C3 write separated by spaces '= SUM ( A1 , A2 )'.

7) You can add chart by clicking on chart Button. There are only 2 charts available (bar, line).
